# 🎓 Student Academic Performance Prediction (FCM + ML)

This project predicts student academic performance using Fuzzy Cognitive Maps (FCM) and machine learning techniques.

## 🚀 Live Demo

👉 https://student-performance-fcm-ml.streamlit.app/

## 📂 Project Files

* `app.py` → Streamlit web app
* `model.py` → ML model logic
* `dataset.csv` → Dataset used
* `requirements.txt` → Dependencies

## 🛠️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Features

* Interactive sliders for input
* Real-time prediction
* Risk classification (Low / Medium / High)
